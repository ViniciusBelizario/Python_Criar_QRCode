# QR Code Generator

Este projeto gera um QR Code a partir de uma string de dados fornecida.

## Requisitos

- Python 3.x
- Biblioteca `qrcode`

   ```sh
   git clone https://github.com/ViniciusBelizario/Python_Criar_QRCode.git
   cd seu-repositorio

2. Instala biblioteca
    ```sh
    pip install -r requirements.txt
